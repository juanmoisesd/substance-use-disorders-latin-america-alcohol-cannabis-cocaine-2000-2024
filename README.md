## Overview

Comprehensive English-language database on substance use disorders and illicit drug consumption across **14 Latin American countries** (2000–2024).

**Author:** Juan Moisés de la Serna Tuya  
**ORCID:** [0000-0002-8401-8018](https://orcid.org/0000-0002-8401-8018)  
**OSF Repository:** [https://osf.io/tcxpr/](https://osf.io/tcxpr/)

## Contents

Country-level data on:

1. **Alcohol use disorder prevalence** (12-month, adults 15–64)
2. **Past-year cannabis use rates** (%)
3. **Past-year cocaine use** (%)
4. **Opioid use disorder prevalence** (per 100,000)
5. **Treatment coverage** (%)
6. **Drug-related mortality** per 100,000
7. **Disability-Adjusted Life Years (DALYs)** from substance use disorders
8. **Longitudinal DALY trend series** 2000–2019

## Countries Covered (14)

Argentina, Brazil, Chile, Colombia, Mexico, Peru, Bolivia, Ecuador, Venezuela, Uruguay, Paraguay, Costa Rica, Guatemala, Honduras

## Key Findings

- **102% increase** in substance use disorder DALYs since 2000
- Opioid incidence **+398%**
- Alcohol remains the primary substance of concern across the region

## Data Sources

- **UNODC** World Drug Report (2022–2025)
- **PAHO** country profiles
- **CICAD/OEA**
- National drug surveys: SEDRONAR (Argentina), INPAD (Brazil), SENDA (Chile)
